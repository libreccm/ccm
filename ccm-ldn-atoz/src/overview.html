Purpose

The ccm-ldn-atoz package enables a site to present all it's content in alphabetical order as an additional service to it's users. Usually the theme provides a link to the page as part of its static elements.

Status

Application is supported by the default internal theme, as well as by the APLAWS standard theme.

Currently only English is available for public site and administration UI.

Recommendations

 

Additional Application Details

 

Usage

By default the application presents the A-Z list at ~/ccm/atoz/.

An administration interface is provided at ~/ccm/atoz/admin/. Here you can configure the content to be listed by selecting an appropriate content provider.

Currently available providers are:

  • ESD Toolkit Domain Provider
  • Item Provider
  • SiteProxy Provider
  • Catgegory Provider
