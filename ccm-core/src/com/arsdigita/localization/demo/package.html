Example A demonstration of how to localize the login and registration pages of the ACS without touching the code. Localization uses German as the target locale, and includes three files:

  • a properties file to contain the localized string (src/ui/login/SubsiteResources_de.properties)
  • a localized stylesheet (xsl/login_de.xsl)
  • an Initializer class (src/localization/demo/Initializer.java) to map the stylesheet to the German locale
