Contains the set of classes that implements the user interface for managing a single {@link com.arsdigita.cms.ContentItem}.

An overview of the roles admin UI

The item admin UI allows a user with the appropriate permissions to perform the following administrative actions on the content item:

  • View a summary of the content item
  • Edit the item using the {@link com.arsdigita.cms.AuthoringKit}
  • Manage the language instances of the top-level content items
  • Administer and manage workflow assignments and tasks
  • Complete workflow assignments
  • Schedule a lifecycle for deployment
  • Manage the version history of the content item
