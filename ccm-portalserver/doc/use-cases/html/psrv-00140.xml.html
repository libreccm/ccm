Use Case 00140:Change Name of Existing Portal Tab

Release Target: 2.0

Description:

This Use Case describes the procedure for changing the name of an existing Portal tab.

Goal Level: user_goal
Actors:
  • Portal Administrator
Trigger: a A PA is unhappy with the name of one of the Portal Tabs, and wishes to modify the name
Preconditions:
Basic Path:


  1. The PA navigates to the Portal and selects the "Configure Portal" link.
  2. The PA is presented with a list of all tabs associated with this Portal
  3. The PA selects the tab that he wishes to rename.
  4. The PA selects the "Rename Tab" link.
  5. A small form is presented to the PA with a field for the tabs name.
  6. The PA changes the name within the Text Field
  7. The PA then clicks the "Save Configuration" button.
Postconditions:
Succesful End Conditions:

The name of the Portal Tab is changed successfully.
