The forum package provides electronic bulletin board where users can post and reply to messages. It is highly integrated with ccm systems user and permission administration as well as search feature. More than one instance of a forum is supported, each with its own set f configuration. There exists a enhancement package forum-categorized which allows a forum to be integrated into navigation system.

Several features and funcionality may be configured (some system wide, most of them on a per forum base), among others:

  • Reply functionality may be disabled so it is just a noticeboard.
  • Users may register for notifications of new posts to the system

Forum.java is the main domain class.
